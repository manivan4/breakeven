# BreakEven Product Brief

**One-liner**: Relationship finance toolkit that helps women split fairly, separate safely, and plan an exit with clear steps—powered by simulated banking data (Nessie) and AI explanations.

## Target Users
- Women who want more control over shared finances
- Women who want a personal buffer while staying in a relationship
- Women who want a step-by-step plan to move out safely

## Modes
- **BALANCE**: fairness + split analysis
- **SEPARATE**: mine/ours routing rules + personal buffer
- **EXIT**: leaving plan with timeline and buffer

## Plans
- **FAST**: aggressive cuts, fastest timeline
- **STEADY**: sustainable, preserves joy budget

## User Flow (MVP)
1. Connect or load demo data (Nessie fallback baked in)
2. Choose mode
3. Set goal amount + current buffer
4. Generate plans (Fast + Steady)
5. Follow weekly checklist and track progress
6. Export or share summary (stretch)

## Key Features (MVP)
- Demo transactions + Nessie connector stub
- Goal setup (exit buffer or personal buffer)
- Deterministic plan generator (no AI for math)
- AI-style explanations and weekly checklist copy (rule-based offline stub)
- Privacy notice + simulated data banner

## UX Principles
- Calm, private, non-judgmental tone
- Action-first: each screen answers “what do I do next?”
- Minimal inputs; defaults come from demo data

## Reusable Components
ModeCard, PlanCard, ProgressBar, CategoryDeltaRow, ChecklistItem, PrivacyNoticeBanner.
